 Dataset retirado do kaggle: [Cardiovascular Disease dataset](https://www.kaggle.com/sulianova/cardiovascular-disease-dataset) 

# Projeto
- No projeto, foi desenvolvido um modelo que prevê casos de problemas cardiovasculares, com base em alguma características de cada indivíduo, como idade, altura, peso, pressão arterial, entre outros.

# Bibliotecas Utilizadas
- Pandas
- Seaborn
- Scikit-learn
